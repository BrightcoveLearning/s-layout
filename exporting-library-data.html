Introduction

The Layout -> App settings -> Data export -> Universal search data functionality is used for the following tasks:

  • Create a Apple UMC Catalog CSV and/or Apple UMC Availability CSV template(s) that generates a data report that is submitted to Apple to enable universal search.
  • Create a Google site map XML template that generates a data report that is used with Google search.
  • Create a Sitemap index XML template that generates a data report that is used as a sitemap.
  • Create a Roku Search Feed 2.0 template that generates a data report that is submitted to Roku to enable universal search.
