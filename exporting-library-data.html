Introduction

The Layout -> App settings -> Data export -> Universal search data functionality is used for the following tasks:

  • Create and Apple UMC Catalog CSV and/or Apple UMC Availability CSV template(s) that generates a data report that is submitted to Apple to enable universal search.
  • Create a Google site map XML template that generates a data report that is used with Google search.
  • Create a Sitemap index XML template that generates a data report that is used as a sitemap.
  • Create a Roku Search Feed 2.0 template that generates a data report that is submitted to Roku to enable universal search.

The basic steps to create one of these data reports is:

  1. Add a new template
  2. Create a new job that uses the template to create a data report.

Adding a new template

To add a new template follow these steps:

  1. In Studio, select Layout, then click the gear icon to enter the app settings.
  2. From App settings, under Data export click Universal search data.
  3. Be sure Configure templates is selected, then click Add new template.
