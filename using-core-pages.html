Introduction

Core pages provide you with options you wish to provide your viewers. Core pages provide:

  • Functionality for viewers to configure, and view information about, their app instance.
  • Basic services like sign in, exit, logout and search.
  • Lists of past viewer activities like recently watched, purchases, downloads and videos marked as a favorite.
  • Display live tv and EPG (electronic program guide) schedules.

A subset of all the core pages are added to every {{ site.product_short }} instance, and others added through onboarding and as needs arise.

For all core pages you can edit the actual name of the page (in all supported languages) and the page's availability. This is reflected in the following table's Confugurable column. For core pages that you can configure beyond those two features, there is a link in the table which will take you to configuration details.

Configuring Favorites

When configuring the Favorites core page the first decision to make is if you want all videos marked as a favorite organized by asset type. If yes, the asset types will be listed across the top of the favorites as clickable links to show only a specific asset type. If no, all the favorites will be grouped together. The following screenshots show first the organized by asset type layout and then the non-grouped layout.

Other configuration options are:

  • If you choose to organize by asset type, you can then set for each type if the thumbnails shown should be shown as landscape or portrait and if the video title should be displayed.
  • If you do not group, you still have those display configuration options, but all thumbnails will follow the selected configuration.
  • If you choose the portrait thumbnails you can also choose if the title position, if displayed, should be inside or outside the thumbnail.
  • As with all core pages, you can also configure the Favorites page's name and availability.

Configuring My List

The My List page provides the viewer with a single page from which they can select to see Recently Watched, Purchases or Favorites videos. Within the My List page configuration each grouping of videos is configurable as described in this section.

As with all core pages, you can also configure the My List page's name and availability.

Configuring Program Guide

The Program Guide page displays your live TV schedule based on the submitted EPG. You can configure the following:

  • Genres display
  • Number of days forward to display: 5 or 10
  • Number of days backward to display: 0 or 5

As with all core pages, you can also configure the Program Guide page's name and availability.

Configuring Recently Watched

The following guide what is in the Recently Watched list:

  • A video is considered "watched" once it has been viewed for 30 seconds.
  • The list is ordered by most recently watched.
  • A time based system configurable setting determines when videos are removed from the list.

When configuring the Recently Watched core page the first decision to make is if you want all videos that match recently watched criteria by asset type. If yes, the asset types will be listed across the top of the recently watched as clickable links to show only a specific asset type. If no, all recently watched videos will be grouped together. The following screenshots show first the organized by asset type layout and then the non-grouped layout.
